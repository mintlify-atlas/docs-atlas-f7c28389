## How Lists Are Maintained

<Steps>
  <Step title="Automated Checking">
    A bot automatically checks all trackers daily for availability and performance
  </Step>
  <Step title="Deduplication">
    Trackers with the same domain or pointing to the same IP address are removed
  </Step>
  <Step title="Sorting">
    Trackers are sorted by popularity and latency (best to worst)
  </Step>
  <Step title="Daily Updates">
    All lists are regenerated and published every day with the latest data
  </Step>
</Steps>

## Choosing the Right List

<Warning>
  For most users, **trackers_best.txt** is recommended. It contains the 20 most reliable trackers and provides excellent performance without overwhelming your torrent client.
</Warning>

### Decision Guide

| Use Case | Recommended List | Reason |
|----------|------------------|--------|
| General torrenting | `trackers_best.txt` | Optimal balance of speed and reliability |
| Maximum peer discovery | `trackers_all.txt` | Access to all 112 available trackers |
| Firewall restrictions | Protocol-specific lists | Only use allowed protocols |
| DNS blocking/censorship | IP address lists | Bypass DNS-level blocking |
| UDP-only networks | `trackers_all_udp.txt` | Many networks only allow UDP |
| Corporate/school network | `trackers_all_https.txt` | HTTPS often passes firewalls |

<Note>
  More trackers doesn't always mean better performance. The `trackers_best.txt` list with 20 carefully selected trackers often performs better than using all 112 trackers.
</Note>

## Format Specification

All tracker lists use a standardized format:

```text
protocol://domain:port/announce

protocol://domain:port/announce

protocol://domain:port/announce
```

<Info>
  Each tracker URL is followed by a blank line. This format is compatible with most BitTorrent clients including qBittorrent, Transmission, Deluge, and more.
</Info>

## Download Locations

Each list is available from three locations:

1. **GitHub Raw** - `https://raw.githubusercontent.com/ngosang/trackerslist/master/[filename].txt`
2. **GitHub Pages Mirror** - `https://ngosang.github.io/trackerslist/[filename].txt`
3. **jsDelivr CDN** - `https://cdn.jsdelivr.net/gh/ngosang/trackerslist@master/[filename].txt`

<Tip>
  Use the jsDelivr CDN for the fastest downloads with global edge caching. Use GitHub Pages if jsDelivr is blocked in your region.
</Tip>
