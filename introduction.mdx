# TrackersList

TrackersList provides automatically updated lists of public BitTorrent trackers that are validated daily to ensure reliability and performance. With over 112 public trackers monitored continuously, you can easily integrate high-quality tracker lists into your BitTorrent applications.

## How it works

TrackersList uses an automated system to maintain high-quality tracker lists:

1. **Continuous monitoring** - A bot checks all trackers for availability and performance
2. **Smart filtering** - Removes duplicates, dead trackers, and those with malfunction issues
3. **Popularity ranking** - Trackers are sorted by popularity and latency from best to worst
4. **Daily updates** - All lists are refreshed daily to ensure you always have working trackers

## Key features

### Multiple access points

Every list is available through three mirrors for redundancy:
- GitHub raw content (primary)
- GitHub Pages (mirror)
- jsDelivr CDN (mirror 2)

### Blacklist management

The project maintains a [blacklist](https://github.com/ngosang/trackerslist/blob/master/blacklist.txt) to filter out:
- Trackers detected by antivirus software
- Malfunctioning trackers
- Fake seed providers
- Duplicate trackers
- Registered-only torrent trackers

### DNS-free options

If you're experiencing DNS problems, IP address versions are available:
- `trackers_best_ip` (20 trackers)
- `trackers_all_ip` (61 trackers)
- `trackers_all_yggdrasil_ip` (3 trackers)

<Note>
  Lists with IP addresses can be shorter because Cloudflare IPs are removed for security reasons.
</Note>

## Special tracker types

<AccordionGroup>
  <Accordion title="WebSocket trackers (WebTorrent)">
    WebSocket trackers are supported by few clients. These enable browser-based torrent clients and real-time communication. Learn more at [webtorrent.io](https://webtorrent.io).
  </Accordion>
  
  <Accordion title="I2P trackers">
    I2P trackers require an [I2P Router](https://geti2p.net/) and a compatible BitTorrent client like qBittorrent, BiglyBT, Tixati, I2PSnark, or libtorrent. [More info](https://geti2p.net/en/docs/applications/bittorrent).
  </Accordion>
  
  <Accordion title="Yggdrasil trackers">
    Yggdrasil trackers require the [Yggdrasil Network Router](https://yggdrasil-network.github.io/) for end-to-end encrypted networking.
  </Accordion>
</AccordionGroup>
