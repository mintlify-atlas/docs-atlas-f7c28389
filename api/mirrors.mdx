## Overview

TrackersList is available through three different mirror sources to ensure maximum availability and global performance. Each mirror serves identical content updated daily.

<CardGroup cols={3}>
  <Card title="GitHub Raw" icon="github">
    Primary source, direct from repository
  </Card>
  <Card title="GitHub Pages" icon="browser">
    Static hosting with global CDN
  </Card>
  <Card title="jsDelivr CDN" icon="server">
    Public CDN with aggressive caching
  </Card>
</CardGroup>

## Caching Behavior

<Tabs>
  <Tab title="Cache Headers">
    ### Response Headers
    
    Each mirror returns different caching headers:
    
    **GitHub Raw:**
    ```http
    Cache-Control: max-age=300
    ETag: "..."
    X-Content-Type-Options: nosniff
    ```
    
    **GitHub Pages:**
    ```http
    Cache-Control: max-age=600
    ETag: "..."
    X-GitHub-Request-Id: ...
    ```
    
    **jsDelivr:**
    ```http
    Cache-Control: public, max-age=86400
    CDN-Cache-Control: public, max-age=31536000
    ETag: "..."
    ```
  </Tab>
  
  <Tab title="Update Propagation">
    ### Time to Propagate
    
    After lists update (3 AM UTC daily), propagation times vary:
    
    <Steps>
      <Step title="GitHub Raw: Immediate">
        Changes appear instantly (< 1 minute)
      </Step>
      <Step title="GitHub Pages: 2-10 minutes">
        Build and deployment time for static site
      </Step>
      <Step title="jsDelivr: 5-60 minutes">
        CDN cache purge and edge propagation
      </Step>
    </Steps>
    
    <Note>
      For the freshest data, use GitHub Raw. For performance, accept jsDelivr's delay.
    </Note>
  </Tab>
  
  <Tab title="Client Caching">
    ### Recommended Client Strategy
    
    <CodeGroup>
    ```python Python with Caching
    import requests
    from datetime import datetime, timedelta
    import pickle
    
    CACHE_FILE = 'trackers_cache.pkl'
    CACHE_DURATION = timedelta(hours=24)
    
    def get_trackers_cached():
        try:
            with open(CACHE_FILE, 'rb') as f:
                cache = pickle.load(f)
                if datetime.now() - cache['timestamp'] < CACHE_DURATION:
                    return cache['trackers']
        except FileNotFoundError:
            pass
        
        # Try mirrors in order of preference
        mirrors = [
            'https://cdn.jsdelivr.net/gh/ngosang/trackerslist@master/trackers_best.txt',
            'https://ngosang.github.io/trackerslist/trackers_best.txt',
            'https://raw.githubusercontent.com/ngosang/trackerslist/master/trackers_best.txt'
        ]
        
        for mirror in mirrors:
            try:
                response = requests.get(mirror, timeout=10)
                response.raise_for_status()
                trackers = [line.strip() for line in response.text.split('\n') if line.strip()]
                
                # Cache the result
                with open(CACHE_FILE, 'wb') as f:
                    pickle.dump({'timestamp': datetime.now(), 'trackers': trackers}, f)
                
                return trackers
            except requests.RequestException:
                continue
        
        raise Exception('All mirrors failed')
    ```
    
    ```javascript JavaScript with Caching
    const CACHE_KEY = 'trackerslist_cache';
    const CACHE_DURATION = 24 * 60 * 60 * 1000; // 24 hours
    
    async function getTrackersCached() {
      // Check localStorage cache
      const cached = localStorage.getItem(CACHE_KEY);
      if (cached) {
        const { timestamp, trackers } = JSON.parse(cached);
        if (Date.now() - timestamp < CACHE_DURATION) {
          return trackers;
        }
      }
      
      // Try mirrors with failover
      const mirrors = [
        'https://cdn.jsdelivr.net/gh/ngosang/trackerslist@master/trackers_best.txt',
        'https://ngosang.github.io/trackerslist/trackers_best.txt',
        'https://raw.githubusercontent.com/ngosang/trackerslist/master/trackers_best.txt'
      ];
      
      for (const mirror of mirrors) {
        try {
          const response = await fetch(mirror, { signal: AbortSignal.timeout(10000) });
          const text = await response.text();
          const trackers = text.split('\n').filter(line => line.trim());
          
          // Cache the result
          localStorage.setItem(CACHE_KEY, JSON.stringify({
            timestamp: Date.now(),
            trackers
          }));
          
          return trackers;
        } catch (error) {
          continue;
        }
      }
      
      throw new Error('All mirrors failed');
    }
    ```

## Failover Implementation

<Steps>
  <Step title="Define Mirror Priority">
    List mirrors in order of preference based on your use case:
    
    ```javascript
    const mirrors = [
      'https://cdn.jsdelivr.net/gh/ngosang/trackerslist@master/trackers_best.txt',  // Fastest
      'https://ngosang.github.io/trackerslist/trackers_best.txt',                    // Reliable
      'https://raw.githubusercontent.com/ngosang/trackerslist/master/trackers_best.txt' // Authoritative
    ];
    ```
  </Step>
  
  <Step title="Implement Timeout">
    Set reasonable timeouts (5-10 seconds) for each request
  </Step>
  
  <Step title="Try Next Mirror on Failure">
    On timeout or error, immediately try the next mirror
  </Step>
  
  <Step title="Log and Monitor">
    Track which mirrors are failing to detect infrastructure issues
  </Step>
</Steps>

<CodeGroup>
```go Go Failover Example
package main

import (
    "context"
    "fmt"
    "io"
    "net/http"
    "time"
)

func fetchWithFailover(endpoint string) (string, error) {
    mirrors := []string{
        "https://cdn.jsdelivr.net/gh/ngosang/trackerslist@master/",
        "https://ngosang.github.io/trackerslist/",
        "https://raw.githubusercontent.com/ngosang/trackerslist/master/",
    }
    
    client := &http.Client{
        Timeout: 10 * time.Second,
    }
    
    for _, mirror := range mirrors {
        url := mirror + endpoint
        ctx, cancel := context.WithTimeout(context.Background(), 10*time.Second)
        defer cancel()
        
        req, err := http.NewRequestWithContext(ctx, "GET", url, nil)
        if err != nil {
            continue
        }
        
        resp, err := client.Do(req)
        if err != nil {
            continue
        }
        defer resp.Body.Close()
        
        if resp.StatusCode == 200 {
            body, err := io.ReadAll(resp.Body)
            if err == nil {
                return string(body), nil
            }
        }
    }
    
    return "", fmt.Errorf("all mirrors failed for %s", endpoint)
}
```

```rust Rust Failover Example
use reqwest;
use std::time::Duration;

async fn fetch_with_failover(endpoint: &str) -> Result<String, Box<dyn std::error::Error>> {
    let mirrors = vec![
        "https://cdn.jsdelivr.net/gh/ngosang/trackerslist@master/",
        "https://ngosang.github.io/trackerslist/",
        "https://raw.githubusercontent.com/ngosang/trackerslist/master/",
    ];
    
    let client = reqwest::Client::builder()
        .timeout(Duration::from_secs(10))
        .build()?;
    
    for mirror in mirrors {
        let url = format!("{}{}", mirror, endpoint);
        
        match client.get(&url).send().await {
            Ok(response) if response.status().is_success() => {
                match response.text().await {
                    Ok(text) => return Ok(text),
                    Err(_) => continue,
                }
            }
            _ => continue,
        }
    }
    
    Err("All mirrors failed".into())
}
```

| Mirror | North America | Europe | Asia | Australia |
